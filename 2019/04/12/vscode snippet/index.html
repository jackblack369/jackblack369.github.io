Skill

  • open tab in new window

    • Press Ctrl + k
    • Release the Ctrl and k keys
    • press O (the letter O, not Zero).

  • save all

    • Press Ctrl + k
    • Release the Ctrl and k keys
    • Press s

  • copy line

    Shift+Alt+Down and Shift+Alt+Up

Plugin

  • fileheader

    ctrl+alt+i can insert comments in the head.
