文件系统

  • Mac 默认可以读 Windows 的 NTFS 格式，但不能写。

  • Windows 无法识别 Mac 的 HFS+ 或 APFS 格式。

  • Mac 和 Windows 都能正常读写 FAT32 和 ExFAT 格式
